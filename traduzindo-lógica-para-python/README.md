# Modularizacao

> Projeto desenvolvido por **Anny Isabelly Silva Santos**  
> Curso: Análise e Desenvolvimento de Sistemas — UNICID

---

## Descrição

Separação do código em funções independentes para facilitar manutenção, leitura e reutilização.

---

## O que usei

- Funções e escopo
- Reutilização de código
- Organização modular
- Princípio da responsabilidade única

---

## O que aprendi

- Código modularizado é mais fácil de manter e testar.
- Funções bem nomeadas tornam o código autodocumentado.

---

## Como Executar
1. Clone o repositório.
2. Instale as dependências: `pip install -r requirements.txt`.
3. Execute o arquivo principal: `python main.py`.
